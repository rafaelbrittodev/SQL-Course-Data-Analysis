# SQL-Course-Data-Analysis
Welcome to the SQL queries repository, executed in pgAdmin and PostgreSQL. This project hosts the SQL code created during the Data Analysis course on Udemy taught by Professor Midori Toyota.

What You Will Find Here: This repository contains the SQL queries and scripts developed throughout the course, covering topics from the basics up to advanced data analysis techniques.

This Material is Ideal for:

Reviewing and practicing the SQL concepts and commands taught in the course.

Consulting practical examples of how to apply SQL in various analysis scenarios.

Deepening your understanding of PostgreSQL/pgAdmin syntax and efficient usage.

How to Use: The queries are organized to easily follow along with the course. Simply clone or download the repository and execute the scripts in your own PostgreSQL or pgAdmin instance.

About the Course: This project is a practical complement to the Udemy course:

SQL para Análise de Dados: Do básico ao avançado (SQL for Data Analysis: From Basic to Advanced)

Link: https://www.udemy.com/course/sql-para-analise-de-dados/
